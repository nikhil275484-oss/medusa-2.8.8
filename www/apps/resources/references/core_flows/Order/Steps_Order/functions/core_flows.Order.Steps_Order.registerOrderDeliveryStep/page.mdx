# registerOrderDeliveryStep - Medusa Core Workflows Reference

This documentation provides a reference to the `registerOrderDeliveryStep`. It belongs to the `@medusajs/medusa/core-flows` package.

This step registers a delivery for an order fulfillment.

```ts title="src/workflows/my-workflow.ts"
import { createWorkflow } from "@medusajs/framework/workflows-sdk"
import { registerOrderDeliveryStep } from "@medusajs/medusa/core-flows"

const myWorkflow = createWorkflow(
  "my-workflow",
  () => {
    const data = registerOrderDeliveryStep({
      "order_id": "order_123"
    })
  }
)
```
