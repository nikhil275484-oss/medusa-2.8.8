# archiveOrderWorkflow - Medusa Core Workflows Reference

This documentation provides a reference to the `archiveOrderWorkflow`. It belongs to the `@medusajs/medusa/core-flows` package.

This workflow archives one or more orders. It's used by the 
[Archive Order Admin API Route](https://docs.medusajs.com/api/admin#orders\_postordersidarchive).

You can use this workflow within your customizations or your own custom workflows, allowing you to wrap custom logic around archiving orders.
