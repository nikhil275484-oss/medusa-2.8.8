# createOrderExchanges - Order Module Reference

This documentation provides a reference to the `createOrderExchanges` method. This belongs to the Order Module.

<Note>

You should only use this methods when implementing complex customizations. For common cases, check out [available workflows instead](/medusa-workflows-reference).

</Note>

## createOrderExchanges(data, sharedContext?): Promise&#60;[OrderExchangeDTO](../../../interfaces/order.OrderExchangeDTO/page.mdx)&#62;

This method creates an order exchange.

### Example

```ts
const exchange = await orderModuleService.createOrderExchanges({
  order_id: "123",
  additional_items: [
    {
      id: "123",
      quantity: 1
    }
  ]
})
```
