# cancelExchange - Order Module Reference

This documentation provides a reference to the `cancelExchange` method. This belongs to the Order Module.

<Note>

You should only use this methods when implementing complex customizations. For common cases, check out [available workflows instead](/medusa-workflows-reference).

</Note>

This method cancels an exchange.

## Example

```ts
const exchange = await orderModuleService.cancelExchange({
  exchange_id: "123"
})
```
