# updateOrders - Order Module Reference

This documentation provides a reference to the `updateOrders` method. This belongs to the Order Module.

<Note>

You should only use this methods when implementing complex customizations. For common cases, check out [available workflows instead](/medusa-workflows-reference).

</Note>

## updateOrders(data, sharedContext?): Promise&#60;[OrderDTO](../../../interfaces/order.OrderDTO/page.mdx)[]&#62;

This method updates existing orders. The order IDs are specified in each order object.

### Example

```typescript
const orders = await orderModuleService.updateOrders([{
  id: "123",
  email: "example@gmail.com"
}])
```
