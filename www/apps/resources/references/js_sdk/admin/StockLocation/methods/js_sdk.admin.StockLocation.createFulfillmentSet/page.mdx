# createFulfillmentSet

This method adds a new fulfillment set to a stock location. It sends a request to the
[Add Fulfillment Set to Stock Location](https://docs.medusajs.com/api/admin#stock-locations\_poststocklocationsidfulfillmentsets)
API route.

## Example

```ts
sdk.admin.stockLocation.createFulfillmentSet("sloc_123", {
  name: "Shipping",
  type: "shipping",
})
.then(({ stock_location }) => {
  console.log(stock_location)
})
```
