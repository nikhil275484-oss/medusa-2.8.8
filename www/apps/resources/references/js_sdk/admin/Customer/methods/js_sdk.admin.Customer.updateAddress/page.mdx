# updateAddress

This method updates a customer address. It sends a request to the
[Update Customer Address](https://docs.medusajs.com/api/admin#customers\_postcustomersidaddressesaddressid)
API route.

## Example

```ts
sdk.admin.customer.updateAddress("cus_123", "cus_addr_123", {
  address_1: "123 Main St",
  city: "Anytown",
  country_code: "US",
  postal_code: "12345"
})
.then(({ customer }) => {
  console.log(customer)
})
```
