# update

This method updates a store. It sends a request to the
[Update Store](https://docs.medusajs.com/api/admin#stores\_poststoresid)
API route.

## Example

```ts
sdk.admin.store.update("store_123", {
  name: "My Store",
})
.then(({ store }) => {
  console.log(store)
})
```
