# initiateRequest

This method initiates a return request by creating a return. It sends a request to the
[Create Return](https://docs.medusajs.com/api/admin#returns\_postreturns)
API route.

## Example

```ts
sdk.admin.return.initiateRequest({
  order_id: "order_123",
})
.then(({ return }) => {
  console.log(return)
})
```
