# list

This method retrieves a list of returns. It sends a request to the
[List Returns](https://docs.medusajs.com/api/admin#returns\_getreturns)
API route.

## Example

To retrieve the list of returns:

```ts
sdk.admin.return.list()
.then(({ returns, count, limit, offset }) => {
  console.log(returns)
})
```

To configure the pagination, pass the `limit` and `offset` query parameters.

For example, to retrieve only 10 items and skip 10 items:

```ts
sdk.admin.return.list({
  limit: 10,
  offset: 10
})
.then(({ returns, count, limit, offset }) => {
  console.log(returns)
})
```

Using the `fields` query parameter, you can specify the fields and relations to retrieve
in each return:

```ts
sdk.admin.return.list({
  fields: "id,*items"
})
.then(({ returns, count, limit, offset }) => {
  console.log(returns)
})
```

Learn more about the `fields` property in the [API reference](https://docs.medusajs.com/api/admin#select-fields-and-relations).
