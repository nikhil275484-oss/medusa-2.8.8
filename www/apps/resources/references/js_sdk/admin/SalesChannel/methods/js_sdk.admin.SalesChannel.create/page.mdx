# create

This method creates a new sales channel. It sends a request to the
[Create Sales Channel](https://docs.medusajs.com/api/admin#sales-channels\_postsaleschannels)
API route.

## Example

```ts
sdk.admin.salesChannel.create({
  name: "Storefront",
})
.then(({ salesChannel }) => {
  console.log(salesChannel)
})
```
