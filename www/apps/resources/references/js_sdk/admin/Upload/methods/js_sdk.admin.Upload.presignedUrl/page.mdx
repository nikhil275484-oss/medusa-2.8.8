# presignedUrl

This method creates a presigned URL for a file upload. It sends a request to the
`/admin/uploads/presigned-urls` API route.

## Example

```ts
sdk.admin.upload.presignedUrl({
  name: "test.txt",
  size: 1000,
  type: "text/plain",
}))
```
